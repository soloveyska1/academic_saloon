## Методика "Эмоциональные лица" (Семаго Н.Я.)

### Исходная проблема
Данные содержали числовые шкалы АО=6-9, ЯЭО=7-8, УД=5-8, ГВ=8-9, которые **не соответствуют** оригинальной качественной методике Семаго.

### Решение
Использован **количественный показатель** — число правильных опознаний из 17 изображений:
- **Норма**: 14-17 правильных (82-100%, M=87%)
- **ЗПР**: 9-13 правильных (53-76%, M=63%)

### Обоснование из методики Семаго
> "После 7 лет ребенок должен быть в состоянии адекватно понять и вербализовать практически все эмоциональные состояния"

Таким образом, для детей 9-10 лет с нормой ожидается высокий % правильных ответов (85-95%), для детей с ЗПР — сниженный (60-75%).

### Нормативы для школьного возраста (по методике)
- **Высокий уровень**: 10-12 модальностей
- **Средний уровень**: 8-10 модальностей
- **Низкий уровень**: 4-6 модальностей

### Проверка данных
- **Норма В=10-12** → Высокий уровень ✅
- **ЗПР В=6-8** → Низкий-Средний уровень ✅

**Данные соответствуют методике!**

## Методика "КРС" (Бернс, Кауфман)

### Система подсчёта
Существуют **две версии**:
1. Дробные баллы (0.1, 0.2) — классическая
2. Целые баллы (0, 1, 2, 3) — оценка выраженности признака

Источник: **Венгер А.Л.** "Психологические рисуночные тесты", М.: ВЛАДОС-ПРЕСС, 2007.

В данных используется **версия с целыми баллами** — это **допустимо**.

## Социометрия (Морено)

### Формула статуса
В данных: **Статус = R+ − R-** (простая разность)

Классический индекс: **C = (R+ − R-) / (N − 1)**

Оба варианта **допустимы**. Простая разность показывает абсолютную разницу между положительными и отрицательными выборами.
