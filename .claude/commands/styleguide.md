## Юридические стандарты

### Терминология оферты:
| Термин | Значение |
|--------|----------|
| Авторский Макет | Справочный материал, результат услуг |
| Исполнитель | Сервис Academic Saloon |
| Заказчик | Пользователь, принявший оферту |
| Акцепт | Принятие условий (кнопка/оплата/использование) |
| ТЗ | Техническое задание (описание заказа) |

### Двухслойная архитектура документов:
- **Layer 1** (Summary Cards) — простым языком, для пользователя
- **Layer 2** (Full Legal) — юридически точным языком
- Layer 1 всегда ссылается на Layer 2 через `sectionIndex`
- **Никогда** не допускать противоречий между слоями

### Обязательные правовые элементы:
- Сроки — конкретные, с дублированием прописью: `3 (три) рабочих дня`
- Определения — в кавычках при первом использовании: «Авторский Макет»
- Ссылки на НПА — полное название + номер статьи
- Нумерация — `X.Y.` (раздел.пункт)

### Emoji палитра (юридическое):
```
⚖️ - Правовые вопросы, баланс интересов
📋 - Условия, чек-листы
🔒 - Конфиденциальность, ПДн
🛡 - Гарантии, защита
⚠️ - Риски, предупреждения
📜 - Документы, оферта
```
